# Smart Data Mediator (Java)

Smart Data Mediator is a Spring Boot based data analytics dashboard that allows users to query database using natural language and visualize results in charts.

## Features
- Natural language query processing
- Chart visualization (Bar, Pie, Line)
- MySQL database integration
- Admin dashboard
- Query history
- Authentication system

## Technologies Used
- Java Spring Boot
- MySQL
- HTML CSS JavaScript
- Chart.js
- Hibernate JPA

## How to Run
1. Clone repository
2. Configure MySQL database
3. Run Spring Boot application
4. Open browser: localhost:8080
